## 三次握手
1. 第一次握手 client 向 server 发送报文，server 确认可以接受来自 client 的报文
2. 第二次握手 server 向 client 回复报文，client 确认可以收到来自 server 的报文，以及 server 可以收到自己发送的报文
3. 第三次握手 client 向 server 发送报文，server 确认 client 可以收到自己发送的报文
